Phantom Features

Phantom Kernel is an Android Custom Kernel build by UBER/SaberMod Toolchains. Some useful Phantom features are given as:

For more information on toolchain visit here

  • star Built by using latest kernel sources officially or from CAF
  • star Better CPU & GPU Management
  • star More RAM for use and better management
  • star More Battery Juice to use
  • star More CPU Governors & Hotplugs
  • star CPU Overclocked
  • star GPU Overclocked & Underclocked
  • star Sound Control over device
  • star PowerSuspend driver
  • star DoubleTap2Wake Implemented
  • star Enabled ZRAM for low RAM devices
